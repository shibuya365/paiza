# paiza scraping

paiza のスキルテスト問題を平均点が高い順に並び替えて表示及びテキストに表示する

```

□paiza C ランク問題（初級）『C067:【ハッカー入門コラボ問題】数字の調査』平均 75.96 点
□paiza C ランク問題（初級）『C078:株の売買』平均 76.78 点
```
